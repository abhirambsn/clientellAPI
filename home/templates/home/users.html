Users!

The `Users` table has mainly 2 fields which are `id` and `name`. Another field is added which returns a boolean value `true`, if even a single object from the `Opportunity` table have `amount` greater than 100,000. Otherwise, it return `false`. This field has been named as `amountGreaterThan100000`. The APIView returns at maximum 10 rows from the table in form of JSON for the users to view and for developers to integrate with their projects. The links to access the opportunity table are set in syntax 

`{{link}}/api/users/{page-number}`

The pagination links are provided at the end of the objects with keys `previous-page` and `next-page`. These page links are adjusted with help of `request` object from django using method build_absolute_uri() and conditioning it as per limit of objects per page.

Each object is returned with it's number in display order as the key of the JSON dictionary along with a dictionary containing the necessary fields as the value pair of the key.
